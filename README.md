# OIBSIP_JAVADEVELOPEMENT
Project collection: Java mini-projects for OIBSIP (On-the-Job Industry-Based Internship Program)

Overview

This repository contains five Java projects (small applications) built as part of the OIBSIP Java Development track. Each task is a separate, self-contained console / simple GUI application demonstrating core Java skills, OOP concepts, and basic data structures.

Tasks

Online Reservation System — A simple reservation booking system (e.g., hotel / bus / event) with CRUD operations for bookings and customers.

Number Guessing Game — A console-based guessing game where the user tries to guess a randomly generated number with feedback and attempt limits.

ATM Interface — Simulates ATM operations: PIN verification, balance inquiry, withdrawal, deposit, and mini statement.

Online Examination System — A quiz/exam application with timed tests, multiple choice questions, scoring, and basic result storage.

Digital Library Management — Manage books, students, issue/return operations, search and report generation.
